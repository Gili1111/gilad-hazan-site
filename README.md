# אתר תדמית — גלעד חזן

אתר תדמית (One-Pager) סטטי: פתרונות AI, אוטומציה, בניית אתרים ושינויים ארגוניים לעסקים.
עברית RTL, רספונסיבי מלא, בנוי בצבעי המותג (נייבי `#0E1B33` · טורקיז `#00C2A8` · ענבר `#FFB020`).

## קבצים
```
work_AI/
├── index.html        כל תוכן הדף
├── styles.css        העיצוב (טוקני מותג, רספונסיב, אנימציות)
├── script.js         תפריט מובייל, אנימציות גלילה, טופס יצירת קשר
└── assets/
    ├── gilad-portrait.png   הפורטרט
    ├── og-image.png         תמונת שיתוף לרשתות
    └── favicon-32/180/512.png
```

## צפייה מקומית
פותחים את `index.html` בדפדפן, או מריצים שרת קטן:
```
python -m http.server 5173 --directory work_AI
```
ואז גולשים ל-http://localhost:5173

## העלאה לאוויר (Vercel)
- **הכי קל:** נכנסים ל-https://vercel.com, גוררים את תיקיית `work_AI` לעמוד "New Project" → Deploy. תוך דקה האתר באוויר.
- **דרך ה-CLI:** מתוך התיקייה: `npx vercel` ולעקוב אחרי ההנחיות.
אין שלב build — זה אתר סטטי, Vercel מגיש אותו כמו שהוא.
